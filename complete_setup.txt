STOCK TRADING SIMULATION - Complete Application Package

This package contains a full-stack Java Spring Boot application for stock market trading simulation.

FILES INCLUDED:
=============
✓ pom.xml - Maven configuration with all dependencies
✓ application.properties - Database and app configuration
✓ Entity Models - User, Stock, Transaction, Portfolio
✓ Repositories - JPA data access interfaces
✓ Security Configuration - Spring Security setup
✓ Sample CSV Stock Data - 10 stock files with OHLCV data
✓ README.md - Comprehensive documentation
✓ DEPLOYMENT_GUIDE.md - Step-by-step deployment instructions

WHAT YOU NEED TO ADD:
====================
The following files need to be created to complete the application:

1. SERVICE LAYER (src/main/java/com/stocktrading/service/):
   - UserService.java - User management logic
   - StockService.java - Stock data handling
   - TradingService.java - Buy/sell operations
   - PortfolioService.java - Portfolio calculations

2. CONTROLLERS (src/main/java/com/stocktrading/controller/):
   - AuthController.java - Login/logout
   - DashboardController.java - Main dashboard
   - StockController.java - Stock browsing and charts
   - TradingController.java - Trade execution
   - AdminController.java - Admin panel

3. FRONTEND (src/main/resources/):
   templates/:
     - login.html - Login page
     - dashboard.html - User dashboard
     - stocks.html - Stock listing
     - stock-detail.html - Individual stock with chart
     - portfolio.html - User portfolio
     - admin-dashboard.html - Admin panel
   
   static/css/:
     - style.css - Main stylesheet
   
   static/js/:
     - charts.js - Chart.js integration
     - trading.js - Trading functionality

4. DATA INITIALIZATION:
   - DataLoader.java - Loads users and CSV data on startup

======

2. mvn spring-boot:run
3. Access at http://localhost:8080

The application will automatically:
- Create database tables
- Load 2 admin users + 30 regular users
- Import stock data from CSV files
- Set up security and authentication
