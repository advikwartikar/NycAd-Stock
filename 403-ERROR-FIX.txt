CSRF = Cross-Site Request Forgery protection

Spring Security generates a unique token for each session.
Every POST request MUST include this token.
Without it â†’ 403 Forbidden error.

Thymeleaf automatically provides the token via:
  th:name="${_csrf.parameterName}"
  th:value="${_csrf.token}"

This creates a hidden field like:
  <input type="hidden" name="_csrf" value="abc123..."/>
